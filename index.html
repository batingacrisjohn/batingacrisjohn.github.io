Bachelor of Science in Computer Science

The BS Computer Science program focuses on the design, development, and application of computing systems that solve complex problems and drive technological innovation.

About the Course

The Bachelor of Science in Computer Science (BSCS) is a four-year program that provides students with a strong foundation in programming, algorithms, data structures, and computer systems.

It emphasizes problem-solving, analytical thinking, and innovation in designing computer-based solutions.

The course integrates both theoretical and practical learning experiences â€” from coding and software development to research in emerging technologies such as Artificial Intelligence, Machine Learning, and Human-Computer Interaction (HCI).

Core Subjects

  • Introduction to Computing
  • Data Structures and Algorithms
  • Database Management Systems
  • Operating Systems
  • Software Engineering
  • Artificial Intelligence
  • Human-Computer Interaction
  • Web and Mobile Application Development

Featured Student Projects

AI-Based Crop Monitoring System

A system that uses computer vision and machine learning to detect crop health and optimize agricultural productivity. Supports SDG 2: Zero Hunger.

Cat Skin Disease Detection Using MobileNetV3

A web system that helps veterinarians and pet owners identify cat skin diseases using image recognition and a rule-based diagnosis model.

Web-Based Musical Note Recognition System

A CNN-powered web tool that recognizes and converts musical notes from scanned sheet music into digital notation, enhancing accessibility for musicians.
